# Scope Boundary

## Build

- Telecom/broadband service recovery case workflow.
- Simulated enterprise systems.
- Structured agent interpretation.
- Deterministic policy enforcement.
- Maestro Case routing/human review.
- Eval/usefulness layer.
- Governed improvement artifact.

## Do Not Build

- Real telecom integrations.
- Generic agent governance SaaS.
- LLM-owned final decisions.
- LLM self-mutating production policy.
- Full monitoring dashboard.
- Full Test Cloud product.
- Every telecom failure type.
- Unnecessary UI polish outside demo-critical surfaces.

## Supported Failure Types First

- missing telemetry,
- stale telemetry,
- contradicting telemetry,
- dispatch dependency from technician note,
- inventory/device mismatch,
- adversarial pressure to bypass checks,
- invalid agent output.
