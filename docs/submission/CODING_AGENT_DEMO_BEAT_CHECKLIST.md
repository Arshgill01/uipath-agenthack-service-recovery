# Coding-Agent Demo Beat Checklist

Use this as the 10-20 second coding-agent bonus beat in the final video.

## Show

1. `README.md` coding-agent paragraph.
2. `docs/submission/CODING_AGENT_PROOF_LOG.md` claim and evidence inventory.
3. `docs/submission/coding_agent_evidence_manifest.json` for branch/commit evidence and runtime-authority flags.
4. Terminal output from `scripts/run_submission_check.sh`.

## Say

> Codex helped build, test, validate, and document the solution. UiPath remains the runtime orchestration and governance layer, and deterministic policy owns closure decisions.

## Do Not Say

- Codex closes cases.
- Codex mutates production policy.
- Codex replaces UiPath Maestro Case, Action Center accountability, deterministic policy, or human review.
- Chat logs prove the work. The public proof is git branches, commits, docs, build logs, artifacts, and validation output.
